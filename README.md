# CineDay

 Descrição

CineDay é um sistema de recomendação de filmes desenvolvido em Java, utilizando Programação Orientada a Objetos, princípios SOLID e testes automatizados com JUnit e Mockito.

O sistema recomenda filmes com base nas preferências do usuário, como gênero, duração, idioma e classificação etária.

---

 Tecnologias

* Java 22
* JUnit 5
* Mockito
* UML

---

Estrutura

```text
src
 ├── model
 ├── service
 ├── util
 ├── exceptions
 ├── catalogo
 └── test
```

---

Funcionalidades

* recomendação de filmes;
* cálculo de score;
* filtragem de filmes;
* recomendação aleatória;
* testes unitários.

---

 SOLID

O projeto aplica os princípios SOLID por meio da separação de responsabilidades, uso de interfaces e desacoplamento entre classes.
